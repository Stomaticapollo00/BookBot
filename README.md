# bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!

User initiates program via CLI using command 'bookbot'

Bookbot will ask if you want a book report, ask for a book and then run and return:
    The word count of the book
    The sorted count of each letter in the book by number of appearences

Bookbot is using a downloaded directory of books obtained by scraping the gutenberg.org library of .txt books in english(en)
